### 二、定义词法器
#### 2.1定义一个词法器结构体
```go
type Lexer struct {  
	input string  //输入的词
	position int // 输入的字符串中的当前位置 (指向当前字符)  
	readPosition int // 输入的字符串中的当前读取位置 (指向当前字符之后的一个字符(ch))  
	ch byte // 当前正在查看的字符  
}
```
#### 2.2读下一个字符
readChar是一个读取字符并且能够自动后移的函数。首先需要判断要读的字符是否越界：判断的依据是下一个字符的位置（readPosition）是否大于等于输入字符串input的长度。
- 是，则终止读取将当前读取的字符串置0。
- 否，则将当前读取的字符置为下一个字符（readPosition）
最后，将字符前移。
```go
// 读取input的下一个字符，并前移其在input中的位置
// 检查是否到到input的结尾
func (l *Lexer) readChar() {
	if l.readPosition >= len(l.input) { //下一个要读取的字符位置大于整个输入字符串的长度，已经读到输入的末尾
		l.ch = 0 // NUL的ASSII码(0)，表示尚未读取任何内容或文件结尾
	} else {
		// //未到input结尾，将l.ch置为下一个字符（也就是input对应的下一个要读的字符的位置（readPosition））
		l.ch = l.input[l.readPosition]
	}
	// 前移
	l.position = l.readPosition
	l.readPosition += 1
}
```
初始化Lexer词法分析器
```go
func New(input string) *Lexer {
	l := &Lexer{input: input}
	// 初始化 l.ch,l.position,l.readPosition
	l.readChar()//读取第一个字符
	return l
}
```

读取下一个token
```go
// 根据当前的ch创建词法单元
func (l *Lexer) NextToken() token.Token {
	var tok token.Token
	switch l.ch {
	case '=':
		tok = newToken(token.ASSIGN, l.ch)
	case ';':
		tok = newToken(token.SEMICOLON, l.ch)
	case '(':
		tok = newToken(token.LPAREN, l.ch)
	case ')':
		tok = newToken(token.RPAREN, l.ch)
	case ',':
		tok = newToken(token.COMMA, l.ch)
	case '+':
		tok = newToken(token.PLUS, l.ch)
	case '{':
		tok = newToken(token.LBRACE, l.ch)
	case '}':
		tok = newToken(token.RBRACE, l.ch)
	case 0://置空
		tok.Literal = ""//字面值为空
		tok.Type = token.EOF//文件结尾  
	}
	l.readChar()//读字符 循环向后读
	return tok //返回词法单元
}
```
