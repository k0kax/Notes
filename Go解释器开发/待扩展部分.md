## 一、基础语法类
### 1.条件选择语句
条件选择语句IF扩展为：
```go
if(条件1){
	结果1
}else if(条件2){
	结果2
}else if(条件3){
	结果3
}......
......
else{
	最后结果
}
```

添加switch case语法
```go
witch 条件选择:{
case 条件1：
	语句1;
	break;
case 条件2：
	语句2;
	continue;
.....

default:
	最后语句
}
	
```

实现break跳出本步执行，并跳出循环或者选择，和continue继续进行不跳出选择或循环，但中断
### 2.循环语句
添加 for循环
```go
for(条件1;条件2;条件3){
	循环语句
}
```
添加range函数，进行遍历

### 3.逻辑运算符
```go
&& //与运算
|| //或运算
```

### 4.注释
```go
// 单行注释
/*
	多行注释
*/
```
